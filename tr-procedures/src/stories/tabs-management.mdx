# Tabs Management

This section explains the concepts related to managing tabs in the frontend.
The main purpose for the main tabs is to allow the user to have access in a easy way to certain views without having to open the procedures menu and navigate through all the windows assigned to the user role.


## Concept Overview

Tabs management is crucial for organizing content and providing a better user experience. It involves dynamically adding, removing, and switching between tabs.

### Key Functions

- **addTab**: Function to add a new tab.
- **removeTab**: Function to remove an existing tab.
- **switchTab**: Function to switch between tabs.
