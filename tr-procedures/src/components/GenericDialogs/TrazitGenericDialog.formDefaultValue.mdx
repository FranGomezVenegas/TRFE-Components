The `formDefaultValue` function initializes form fields with default values based on the specified configuration. Below are examples demonstrating different field configurations and how they are handled by the function.

## Options

### 1. default_value property

When fields have a `default_value` property, `formDefaultValue` assigns this value 'as is' to the corresponding form field.

```json
{
  "text1": {
    "label_en": "Parent Code",
    "label_es": "CÃ³digo paterno",
    "default_value": "demo default value"
  }
}
```

### 2. getNextId property

This feature requires the boolean property **getNextId set to true** and **internalVariableObjName** to set the array variable to count the current length and get the next index

(possible enhancement, getNextMaxId. In this case it would be to get the max value in any entry and return it +1)


```json
{
    "number1": {
        "label_en": "Order",
        "label_es": "Orden",
        "getNextId" : true,
        "internalVariableObjName": "selectedItem"
    }
}
```
